# Task Management API - Backend Spring Boot

API RESTful para gerenciamento de tarefas desenvolvida com Spring Boot, JPA/Hibernate e Oracle Database.

- **Java 17** - Linguagem de programação

- **Hibernate** - ORM (Object-Relational Mapping)

- **Maven** - Gerenciamento de dependências
- **JUnit 5** - Testes unitários

##  Funcionalidades da API

### Endpoints Principais

#### Tarefas
- `GET /api/tasks` - Listar tarefas com paginação e filtros
- `GET /api/tasks/{id}` - Buscar tarefa por ID
- `POST /api/tasks` - Criar nova tarefa
- `PUT /api/tasks/{id}` - Atualizar tarefa existente
- `DELETE /api/tasks/{id}` - Excluir tarefa
- `GET /api/tasks/statistics` - Estatísticas das tarefas

### Parâmetros de Consulta

#### Paginação e Ordenação
- `page` - Número da página (padrão: 0)
- `size` - Tamanho da página (padrão: 10)
- `sort` - Campo para ordenação (padrão: dataCriacao)
- `direction` - Direção da ordenação: ASC/DESC (padrão: DESC)

#### Filtros
- `titulo` - Filtro por título (busca parcial)
- `status` - Filtro por status (PENDENTE, EM_ANDAMENTO, CONCLUIDA)
- `usuarioAtualizacao` - Filtro por usuário (busca parcial)

### Exemplo de Requisições

```bash
# Listar todas as tarefas
GET /api/tasks

# Listar com paginação e filtros
GET /api/tasks?page=0&size=5&sort=titulo&direction=ASC&status=PENDENTE

# Criar nova tarefa
POST /api/tasks
Content-Type: application/json
{
  "titulo": "Nova Tarefa",
  "descricao": "Descrição da tarefa",
  "status": "PENDENTE",
  "usuarioAtualizacao": "usuario.teste"
}

# Atualizar tarefa
PUT /api/tasks/1
Content-Type: application/json
{
  "titulo": "Tarefa Atualizada",
  "descricao": "Nova descrição",
  "status": "EM_ANDAMENTO",
  "usuarioAtualizacao": "usuario.teste"
}
```

## ️ Modelo de Dados

### Entidade Task
```sql
CREATE TABLE tasks (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    titulo VARCHAR2(255) NOT NULL,
    descricao VARCHAR2(1000) NOT NULL,
    status VARCHAR2(20) NOT NULL CHECK (status IN ('PENDENTE', 'EM_ANDAMENTO', 'CONCLUIDA')),
    data_criacao TIMESTAMP DEFAULT CURRENT_TIMESTAMP NOT NULL,
    data_atualizacao TIMESTAMP DEFAULT CURRENT_TIMESTAMP NOT NULL,
    usuario_atualizacao VARCHAR2(100) NOT NULL
);

CREATE SEQUENCE task_sequence START WITH 1 INCREMENT BY 1;
```

## Configuração e Execução

### Pré-requisitos
- Java 17+
- Maven 3.6+

### Configuração do Banco Oracle

1. **Criar usuário e schema:**
```sql
CREATE USER taskmanager IDENTIFIED BY password;
GRANT CONNECT, RESOURCE TO taskmanager;
GRANT CREATE SESSION TO taskmanager;
GRANT CREATE TABLE TO taskmanager;
GRANT CREATE SEQUENCE TO taskmanager;
```

2. **Configurar connection string no application.yml:**
```yaml
spring:
  datasource:
    url: jdbc:oracle:thin:@localhost:1521:XE
    username: taskmanager
    password: password
```
